📌 About the Project

BullyWatch is a smart wearable system designed to help students report bullying incidents quickly. It connects a wearable device to an application that alerts nearby trusted users in real time.

⚠️ Problem

Many bullying victims cant ask for help immediately due to fear.

⚙️ How It Works

1. Press panic button
2. Send alerts
3. Notify nearby users
4. Help arrives quickly (hopefully)

🚀 Features

  • Hidden panic button
  • Real-time alerts
  • GPS location
  • Silent mode
