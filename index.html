BullyWatch

Smart Wearable Anti-Bullying Alert System

📌 About the Project

BullyWatch is a smart wearable system designed to help students report bullying incidents quickly and discreetly. It connects a wearable device to a mobile application that alerts nearby trusted users in real time.

🎯 Problem

Many bullying victims cannot ask for help immediately due to fear or isolation. Bystanders also often fail to respond quickly because they are unaware of the situation.

⚙️ How It Works

1. User presses hidden panic button on wearable device
2. Alert is sent to mobile app
3. Nearby teachers/students receive notification
4. Help responds immediately

🚀 Key Features

  • Hidden emergency button
  • Real-time alerts
  • GPS location tagging
  • Silent activation mode
  • Teacher & guardian notifications

🎓 Purpose

To improve school safety by using technology to create faster response systems for bullying incidents.
